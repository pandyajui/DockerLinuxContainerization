Linux Commands

  • pwd command
    • The pwd command is to find out the path of the current working directory (folder) you’re in. The command will return an absolute (full) path, which is basically a path of all the directories that starts with a forward slash (/). An example of an absolute path is /home/username.
  • cd command
    •  To navigate through the Linux files and directories, we use the cd command. It requires either the full path or the name of the directory, depending on the current working directory that you’re in. An Example: cd Desktop
  • ls command
    •  The ls command is used to view the contents of a directory. By default, this command will display the contents of your current working directory.
  • cp command
    •  Use the cp command to copy files from the current directory to a different directory. For instance, the command cp rock.jpg /home/username/Pictures would create a copy of rock.jpg (from your current directory) into the Pictures directory.
  • mv command
    •  The primary use of the mv command is to move files, although it can also be used to rename files.
    Move Example: mv /home/prithvi/test /home/prithvi/Downloads/
    Rename Example: mv test test2
  • mkdir command
    •  Use mkdir command to make a new directory — if you type mkdir Books it will create a directory called Books.
  • rmdir command
    •  If you need to delete a directory, use the rmdir command. However, rmdir only allows you to delete empty directories.
    Example: rmdir Books
  • rm command
    •  The rm command is used to delete directories and the contents within them.
    Example: rm -d Downloads
    Note: Be very careful with this command and double-check which directory you are in. This will delete everything and there is no undo.
  • grep command
    •  Another basic Linux command is grep. It is used to search for a string of characters in a specified file.
    Example: grep phoenix sample2
  • sudo command
    •  Short for “SuperUser Do”, this command enables you to perform tasks that require administrative or root permissions. However, it is not advisable to use this command for daily use because it might be easy for an error to occur if you did something wrong.
    Example: sudo apt-get update
  • chmod command
    •  chmod is another Linux command, used to change the read, write, and execute permissions of files and directories. As this command is rather complicated, you can read the full tutorial in order to execute it properly.
